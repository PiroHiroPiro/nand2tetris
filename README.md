# nand2tetris

A repository for learning [nand2tetris](https://www.nand2tetris.org/).

## Requirement

- Python:3.6 or later
- Pipenv:2018.11.26 or later

## Install

Clone repository:

```console
$ git clone https://github.com/PiroHiroPiro/nand2tetris.git
$ cd nand2tetris
```
